# Home

Home a [Sails](http://sailsjs.org) application for my home :)

## Dependencies:

Build with Sails.js and Angular.js (backend/frontend),
MongoDB was used as database


# Install

Installing the backend:
```
cd home/
npm install
```

Installing the frontend:
```
cd home/assets/
npm install
```
# Start
```
sails lift
```
